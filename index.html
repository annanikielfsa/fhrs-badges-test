FHRS front-end - new badges test

Code snippet

<script src="https://fhrsfrontend.azurewebsites.net/embed/embed-badge.js" data-business-id="1214168" data-rating-style="3" data-welsh="false"></script>

  • data-business-id is the business' unique id
  • data-rating-style is one of badge designs - values 1 to 5 work, if I put 6, some of the badges dissapear. Only 1-3 options are shown on the ratings website.
  • data-welsh - if true, bilingual badge is displayed. This can be switched on and off for English / Welsh businesses (same as in current badges)
